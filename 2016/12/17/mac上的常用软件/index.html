Mac常用软件

总览

软件

开发类

Beyond Compare

老牌diff软件，偶尔数据清洗之后用它看看效果

Eclipse

不思进取越来越差的典型，不用，也懒得删

Xcode

折腾swift和touch bar的时候就打开，感觉本体还不如 Xcode 命令行有存在感。（touch bar这次更新终于有了官方中文名字：搓擦条…
库克，这是爸爸送你的：

ULtra Edit

什么文件都可以打开看看里面是啥，上次检查用fpdf生成可以看但是不能正常保存的pdf时候立功了，平时吃灰

Pycharm

比较频繁用的Python IDE,启动速度在mac上可以接受，在普通机械硬盘上不可以接受
功能齐全，该有的都有

PhpStorm

Php IDE,和楼上是一家，所以其他地方一样

CornerStone

SVN软件，使用频率中等

Navicat Premium

数据库管理工具，支持各种数据库类型的各种操作

Anaconda Navigator

学院派的python工具
conda建Python虚拟环境 安装各种包比virtualenv和pip更好用
里面的Ipython，Jupyter工具非常好用

RapidWeaver 7

无编码快速做网页的工具，吃灰中

Virtual Studio Code

VS code 开源 支持各种语言、插件和主题。如果上来就用的它，应该会很经常使用

Mweb

专业的 Markdown 写作、记笔记、静态博客生成软件，吃灰

Textnut

支持MarkDown，爪机也有所以同步也方便，可惜竞品太多，吃灰

Sublime Text3

小巧 扩展丰富 常用

Atom

玩github的不来一个吗

Markdown Plus

Markdown软件，左侧编辑 右侧即可看到显示效果，比如现在就在用

Bear

刚出来就上了APP Store推荐的notes app，颜好，支持Markdown

Ulysses

爪机可以和本子同步，专注写字时候最舒服的软件，

MacVim

完全没有存在感的东西，命令行直接vim，从不用专门打开它，吃灰

扩展插件

F.lux

护眼用，自动根据时间调节显示，除了安装之后经常要面对日薄西山一样颜色的屏幕，别的没毛病

Screenie

可以手动截图的小插件，还可以设置截图位置，不让截图堆满毫无存在感的桌面

TouchBarServer

github上找的TouchBar应用，可以在老本子上也可以使用新款的bar，除了图个新鲜，想不到不吃灰的理由

Paste

颜值很高的粘贴板应用，可以存储大量粘贴历史并随处使用，必备品

Dropshelf

优化了拖文件时候的体验，可以直接把文件拖到任意屏幕边缘固定好，并在该位置随意取用

搜狗输入法

没啥好说的，难道用百度的？

Day One

日记本，可以和爪机上同步，小巧+功能强大

Theine

想让屏幕常亮或者亮固定一段时间？点一下就可以了

Popclip

在任何位置选中一段文字之后，都会弹出来的扩展小插件，可以快速复制粘贴谷歌bing保存到大象或者发微博 etc

Boom2

音质调节软件。习惯之后，如果某一次不用它了，你会受不了你的耳机的

Alfred

mac上最强效率软件，快速干任何事情都离不开它

Folder color

随意调整文件夹图标的小软件，可以让常用文件夹更加醒目

Go2shell

实现 在该位置打开终端 的功能，好用

Manico

任务切换从未如此简单，比macos原生的切换功能更好用，option+快捷键直接打开你想打开的各种软件，节省了不知道多少时间

Moom

我也不知道一个快速调节软件窗口的小插件为什么这么贵，反正100次试用之后就直接入手了付费版，有毒

Monity

通知栏小工具，可以查看CPU 内存 硬盘 电池用量和网络连接等，还能看ip

Bartender 2

各种插件挤满了右上角？用它就可以告别头疼，也就是图上的那个服务员图标，节省空间全靠它
